# KhawarAI – AI Chatbot

A production-ready AI chatbot powered by Google Gemini with real-time streaming responses, user authentication, and persistent chat history.

## Features
- 🤖 Google Gemini 1.5 Flash AI
- 🔐 Supabase Auth (email/password)
- 💾 Chat history saved per user
- ⚡ Real-time streaming responses
- 🎯 4 AI Modes: General · Code · Creative · Translator
- 📱 Fully responsive (mobile + desktop)
- 🌙 Dark theme UI

## Tech Stack
Next.js 14 · React · Tailwind CSS · Supabase · Google Gemini API · Vercel
